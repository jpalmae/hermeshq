# 🌍 Códigos de Idioma Soportados por Gamma.app

Gamma.app soporta 60+ idiomas para generación de contenido.

## 🌎 Idiomas Principales

| Idioma | Código | Región |
|--------|--------|--------|
| **Español (Latinoamérica)** | `es-419` | Latinoamérica |
| Español (España) | `es-es` | España |
| Español (México) | `es-mx` | México |
| Inglés (US) | `en` | Estados Unidos |
| Inglés (UK) | `en-gb` | Reino Unido |
| Inglés (India) | `en-in` | India |
| Portugués (Brasil) | `pt-br` | Brasil |
| Portugués (Portugal) | `pt-pt` | Portugal |
